# تحلیل ریتینگ فیلم‌ها

این پروژه تحلیل داده‌های ریتینگ کاربران برای فیلم‌ها است.  
هدف پروژه: بررسی آماری دیتاست، پاک‌سازی داده‌های ناقص و استخراج کاربران و فیلم‌های پرامتیاز.

---

## درباره پروژه

این کد قابلیت‌های زیر را دارد:

- خواندن داده‌های کاربران و فیلم‌ها از فایل CSV
- پاک‌سازی دیتاست جدید از سطرهای دارای مقادیر خالی (NaN)
- پیدا کردن سطرهای حذف‌شده و بررسی اینکه آیا در دیتاست اولیه هم وجود دارند یا خیر
- محاسبه آمار کلیدی:
  - تعداد کل ریتینگ‌ها
  - تعداد کاربران و فیلم‌ها
  - میانگین، حداقل و حداکثر امتیاز
  - کاربر با بیشترین ریت
  - فیلم با بیشترین ریت
- نمایش نتایج در قالب چاپ ساده در کنسول
- ذخیره خروجی‌ها در سه فایل CSV:
  - دیتاست پاک‌شده
  - سطرهای حذف‌شده
  - سطرهای حذف‌شده مشترک با دیتاست اولیه

---

## فایل‌ها

- `app.py` : اسکریپت پایتون اصلی برای تحلیل داده‌ها  
- `Dataset.csv` : دیتاست اولیه  
- `dataset1.csv` : دیتاست جدید (قابل پاک‌سازی)  
- `Dataset_Cleaned.csv` : خروجی دیتاست پاک‌شده  
- `Removed_Rows.csv` : خروجی سطرهای حذف‌شده  
- `Common_Removed.csv` : خروجی سطرهای حذف‌شده مشترک با دیتاست اولیه  
- `.vscode/` : تنظیمات VS Code (اختیاری)
