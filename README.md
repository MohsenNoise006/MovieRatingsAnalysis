# Movie Ratings Analysis

این پروژه تحلیل داده‌های ریتینگ کاربران برای فیلم‌ها است.

## درباره پروژه

- خواندن داده‌های کاربران و فیلم‌ها از فایل CSV
- محاسبه آمار کلیدی:
  - تعداد کل ریتینگ‌ها
  - تعداد کاربران و فیلم‌ها
  - میانگین، حداقل و حداکثر امتیاز
  - کاربر با بیشترین ریت
  - فیلم با بیشترین ریت
- نمایش نتایج در قالب جدول

## فایل‌ها

- `app.py` : کد پایتون برای تحلیل داده‌ها
- `Dataset.csv` : فایل دیتاست
- `.vscode/` : تنظیمات VS Code (اختیاری)

## نحوه اجرا

1. مطمئن شوید Python و pandas نصب شده‌اند.
2. مسیر فایل CSV را در `app.py` بررسی کنید.
3. کد را اجرا کنید:

```bash
python app.py
